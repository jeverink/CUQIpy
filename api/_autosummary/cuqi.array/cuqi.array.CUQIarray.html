CUQIarray#

class cuqi.array.CUQIarray(input_array, is_par=True, geometry=None)#

A class to represent data arrays, subclassed from numpy array, along with geometry and plotting

Parameters:

  • input_array (ndarray) – A numpy array holding the parameter or function values.

  • is_par (bool, default True) – Boolean flag whether input_array is to be interpreted as parameter (True) or function values (False).

  • geometry (cuqi.geometry.Geometry, default None) – Contains the geometry related of the data
